# 📄 Currículo — João Pedro Alves Brugnolo

Currículo profissional desenvolvido com **HTML5 e CSS3 puro**, otimizado para visualização no navegador e impressão em uma única página A4.

## ✨ Funcionalidades

- Layout de duas colunas (sidebar + conteúdo principal)
- Design responsivo e elegante com paleta de cores consistente
- Tipografia com **Syne** (títulos) e **Plus Jakarta Sans** (corpo)
- Impressão otimizada via `@media print` — gera exatamente **1 página A4**
- Foto de perfil integrada via URL do LinkedIn
- Links diretos para GitHub, LinkedIn e e-mail

## 🗂️ Estrutura

```
curriculo/
├── index.html       # Estrutura e conteúdo do currículo
└── css/
    └── style.css    # Estilos e layout de impressão
```

## 🚀 Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/brugnoloJoao/curriculo.git
   ```
2. Abra o arquivo `index.html` diretamente no navegador.
3. Para gerar o PDF, pressione `Ctrl+P` (ou `Cmd+P` no Mac) e salve como PDF.

> **Dica de impressão:** selecione a opção *"Sem margens"* e ative *"Gráficos de fundo"* nas configurações de impressão para o melhor resultado.
